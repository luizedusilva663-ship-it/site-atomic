# Atomic Automações - Landing Page Profissional

Bem-vindo ao repositório da landing page da Atomic Automações, desenvolvida para ser uma ferramenta de alta conversão para serviços de automação de WhatsApp para assistências técnicas.

Este projeto foi construído com HTML5, CSS3 puro e JavaScript, focado em performance, SEO, acessibilidade e uma experiência de usuário moderna e futurista.

## Funcionalidades

- **Design Responsivo:** Otimizado para funcionar perfeitamente em qualquer dispositivo (desktop, tablet, mobile).
- **UI/UX Profissional:** Estilo moderno, tecnológico e futurista, com animações suaves e elementos visuais impactantes.
- **Otimização para Conversão:** Estrutura de conteúdo pensada para guiar o visitante à ação principal: contato via WhatsApp.
- **SEO Amigável:** Meta tags configuradas para melhor ranqueamento em motores de busca.
- **Acessibilidade:** Implementação de atributos ARIA e navegação por teclado para garantir que o site seja utilizável por todos.
- **Performance:** Código limpo, CSS e JS externos para carregamento eficiente.
- **Integração com WhatsApp:** Botões de CTA diretos para o WhatsApp, configuráveis.

## Estrutura do Projeto

```
├── assets/
│   ├── atomic-logo.png
│   ├── atomic-logo-footer.png
│   ├── favicon.ico
│   ├── og-image.png
│   └── whatsapp.svg
├── .gitignore
├── index.html
├── style.css
└── script.js
```

- `index.html`: Estrutura principal da landing page com conteúdo e meta tags de SEO.
- `style.css`: Estilos CSS puro, com variáveis para fácil customização de cores e fontes.
- `script.js`: Lógica JavaScript para interatividade (FAQ, animações de scroll).
- `assets/`: Contém todas as imagens, ícones e favicon do projeto.
- `.gitignore`: Arquivo para ignorar arquivos e pastas desnecessárias no Git.

## Como Usar

1.  **Clone o Repositório:**
    ```bash
    git clone [URL_DO_SEU_REPOSITORIO]
    ```
2.  **Personalize:**
    - Abra `index.html` e substitua `SEUNUMERO` pelo seu número de WhatsApp em todos os links.
    - Se desejar, substitua as imagens `atomic-logo.png`, `atomic-logo-footer.png`, `favicon.ico` e `og-image.png` na pasta `assets/` pelas suas próprias imagens.
    - Altere as cores e fontes no `style.css` se necessário.
3.  **Hospedagem no Netlify:**
    - Faça login no Netlify.
    - Clique em "Add new site" -> "Import an existing project".
    - Conecte-se ao seu provedor Git (GitHub, GitLab, Bitbucket).
    - Selecione o repositório onde você clonou este projeto.
    - Configure as opções de deploy (geralmente, o Netlify detecta automaticamente). Certifique-se de que o diretório de publicação é a raiz do projeto (`/`).
    - Clique em "Deploy site".
